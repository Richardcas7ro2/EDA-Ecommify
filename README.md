# Ecommify Database Design

Este repositorio contiene la entrega del Entregable 1 de la Etapa 2 para el diseño conceptual y lógico de la base de datos híbrida del proyecto Ecommify.

## Estructura del Repositorio

De acuerdo con las instrucciones de la Guía de Actividades, el repositorio está estructurado estrictamente de la siguiente manera:

```text
Ecommify_Database_Design/
├── README.md                           # Este archivo de documentación
├── docs/                               # Documentación técnica
│   ├── Documento_Tecnico_Diseno.pdf    # Documento técnico consolidado
│   └── Presentacion_Ejecutiva.pdf      # Presentación resumida del diseño
├── postgresql/                         # Diseño relacional y transaccional
│   ├── schema/                         # Scripts DDL para creación de tablas (ej. 01_ddl_tables.sql)
│   ├── seed_data/                      # Directorio para carga inicial de datos
│   └── queries/                        # Directorio para consultas analíticas
├── mongodb/                            # Diseño documental
│   └── schema/                         # Colecciones de MongoDB (ej. collections_schema.json)
└── notebooks/                          # Notebooks de análisis de datos
    └── Data_Exploration_Analysis.ipynb # Jupyter notebook con el EDA (Análisis Exploratorio)
```
